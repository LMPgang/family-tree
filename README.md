# 家族族谱网站

一个用于展示和管理家族族谱的静态网站，采用纯HTML、CSS和JavaScript开发。

## 项目简介

本项目是一个家族族谱展示网站，旨在记录和传承家族历史，展示家族成员信息和族谱关系。网站包含以下主要功能：

- **首页介绍**：展示家族概况和网站特色
- **族谱树**：可视化展示家族世代关系
- **家族成员**：详细展示每位家族成员的信息
- **关于我们**：介绍家族历史、价值观和成就

## 项目结构

```
族谱项目/
├── index.html          # 首页

├── members.html        # 家族成员页面

├── styles.css          # 全局样式文件
├── script.js           # 主要JavaScript文件
├── data.js             # 族谱数据文件

```

## 数据结构

族谱数据存储在 `data.js` 文件中，采用JSON格式：

```javascript
{
    familyName: "家族名称",
    origin: "籍贯",
    establishedYear: "建立年份",
    description: "家族描述",

    members: [
        {
            id: 1,                          // 成员ID
            name: "姓名",                   // 成员姓名
            generation: 1,                  // 世代
            birthYear: "1850",              // 出生年份
            deathYear: "1920",              // 去世年份（null表示在世）
            birthPlace: "出生地",           // 出生地
            occupation: "职业",             // 职业
            photo: null,                    // 照片路径
            bio: "个人简介",                // 个人简介
            achievements: [],               // 主要成就
            spouse: "配偶姓名",             // 配偶
            children: [],                   // 子女ID数组
            parents: []                     // 父母ID数组
        }
    ]
}
```

## 使用方法

### 本地运行

1. 克隆或下载项目到本地
2. 使用浏览器直接打开 `index.html` 文件
3. 或使用本地服务器（推荐）：

   ```bash
   # 使用Python启动简单HTTP服务器
   python -m http.server 8000

   # 或使用Node.js的http-server
   npx http-server
   ```

4. 在浏览器中访问 `http://localhost:8000`

### 自定义数据

1. 打开 `data.js` 文件
2. 根据实际家族情况修改 `familyData` 对象
3. 添加、修改或删除成员信息
4. 保存文件后刷新浏览器即可看到更新

### 自定义样式

1. 打开 `styles.css` 文件
2. 修改颜色、字体、布局等样式
3. 保存文件后刷新浏览器即可看到效果

## 维护说明

### 更新成员信息
- 编辑 `data.js` 文件中的 `members` 数组
- 确保ID唯一性
- 维护正确的父子关系

### 添加新世代
- 在 `data.js` 中添加新成员
- 设置正确的 `generation` 值
- 更新父子关系

### 备份数据
- 定期备份 `data.js` 文件
- 建议使用版本控制（如Git）
- 保留历史数据版本

## 注意事项

1. **数据安全**：族谱数据包含个人隐私信息，请注意保护
2. **浏览器缓存**：修改数据后可能需要清除浏览器缓存
3. **图片路径**：如添加照片，请确保路径正确
4. **编码格式**：建议使用UTF-8编码保存文件

## 常见问题

### Q: 如何添加新成员？
A: 在 `data.js` 文件的 `members` 数组中添加新对象，设置正确的ID和关系。

### Q: 如何修改网站颜色？
A: 在 `styles.css` 文件中修改CSS变量或颜色值。

### Q: 为什么看不到更新的数据？
A: 请尝试清除浏览器缓存或强制刷新页面（Ctrl+F5）。

### Q: 如何部署到网上？
A: 可以使用GitHub Pages、Netlify、Vercel等免费静态网站托管服务。
