# 家族族谱网站 - 修改指南

## 一、修改家族成员信息（最常用）

打开 `data.js` 文件，找到 `members` 数组，每个成员的格式如下：

```javascript
{
    id: 1,                          // 编号，不要改
    name: "张德厚",                 // 姓名，改成你家族的人名
    generation: 1,                  // 第几代，不要改
    birthYear: "1850",              // 出生年份
    deathYear: "1920",              // 去世年份（在世的写 null）
    birthPlace: "河南省",           // 籍贯
    occupation: "农民",             // 职业
    bio: "家族创始人...",           // 个人简介
    spouse: "李氏",                 // 配偶
    father: 1,                      // 父亲的id编号
    tomb: {                         // 墓地信息（在世的成员没有这个）
        address: "河南省洛阳市...", // 墓地地址
        lat: 34.68,                 // 纬度（地图定位用）
        lng: 112.45,                // 经度（地图定位用）
        image: "https://..."        // 墓地图片网址
    }
}
```

### 常见修改：

| 想改什么 | 改哪个字段 | 举例 |
|---------|-----------|------|
| 改名字 | `name` | `name: "李明"` |
| 改出生年 | `birthYear` | `birthYear: "1990"` |
| 改去世年 | `deathYear` | `deathYear: "2020"` |
| 在世的人 | `deathYear` | `deathYear: null` |
| 改职业 | `occupation` | `occupation: "医生"` |
| 改简介 | `bio` | `bio: "某某某，从事..."` |
| 改墓地地址 | `tomb.address` | `tomb.address: "北京市..."` |

### 添加新成员：

在 `members` 数组最后面，照着上面的格式加一条，注意：
- `id` 要比现有的最大id大1
- `generation` 填第几代
- `father` 填父亲的 `id`

## 五、修改墓地图片

在 `data.js` 中，每个已故成员的 `tomb.image` 是图片网址。

### 方法1：使用网络图片
直接把 `image` 的值改成新的图片网址：
```javascript
image: "https://你的图片网址"
```

### 方法2：使用本地图片
1. 把图片放到项目文件夹里（和 `index.html` 同级）
2. 把 `image` 的值改成：
```javascript
image: "你的图片文件名.jpg"
```

## 六、修改经纬度（地图定位）

在 `data.js` 中，`tomb.lat` 和 `tomb.lng` 决定地图上的位置。

获取经纬度的方法：
1. 打开 https://www.openstreetmap.org
2. 搜索你要标记的地点
3. 在地图上右键点击，选择"显示地址"
4. 网址栏里会显示经纬度，格式如 `#map=15/34.68/112.45`
5. 其中 `34.68` 是纬度(lat)，`112.45` 是经度(lng)

## 九、修改登录账号密码

打开 `auth.js`，找到 `users` 部分：

```javascript
users: {
    "张德厚": "zhang123",    // 用户名: 密码
    "admin": "admin123",     // 管理员账号
}
```

- 添加新账号：照着格式加一行 `"姓名": "密码"`
- 修改密码：直接改引号里的密码
- 删除账号：删掉那一行即可
